# Godisnji

A simple vacation days tracker built with plain HTML, CSS and JavaScript — no build step, no dependencies.

## Features

- Track vacation day allowances per year
- Log vacation periods with automatic weekday calculation
- Carryover support — unused days from a previous year can roll over before a configurable cutoff date
- Results show used vs. allowed days per year with a progress bar
- All data persisted in `localStorage`

## Usage

Open `web-app/index.html` directly in a browser.

No server required.
